About The Game

Brawl-Beasts is an online party game in which you can brawl your friends online as one of your favorite animals in whacky game modes. You can equip different items and skins to differentiate yourself from other players. Become King of the hill, wack some moles and steal eachothers tails to become the best brawl beast in this hilariously silly game. This game uses a randomizer between game modes and maps, this way content can infinitely be added whether it be a new map or a new gamemode.

    • Sound Manager

    The Sound Manager is one of the biggers things i've made in this project. The way it works is it creates the sounds at the start of the game, linking it to a GameObject making it reusable.

    To call the sounds i've made a function called GetSoundByName(string::soundName) which returns Sound. Sound is the script the audio data is being stored in. It contains two methods: Play and Stop. Other than that it also contains the type of audio it is, in this case Master, FX and Music. This info is used to determin the volume based on the users prefered settings.

    This Manager also contains Music Management with sound easing. The way it works is with a function in the Manager called SetCurrentMusic(). This function calculates the steps it has to take for the music to smoothly transition between the previous and current music, and transitions between the two.
